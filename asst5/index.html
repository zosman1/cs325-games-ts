This was especially difficult and took me quite a while just because of how phasers matter engine functions. This was especially hard because the higher dumpsters required more velocity where as if we used that velocity on the lower ones we would miss. After whiteboarding the solution for a while i decided to add a slight velocity modifier based on the angle of cannon. This modifier took a lot tweaking for it to feel somewhat natural. Additionally, the phaser3 matter object has no "accelerate forward" function. It required you to pass in the new X and Y velocity. I was able to solve this using some physics I had learned in high school but JS has no build in sin and cos functions that rely on degrees instead of radiens, which required me to make custom function wrappers. I also needed to develop a way to stop the player from shooting all the bags at once. I was able to create a setTimeout that makes the player wait one second before they're able to shoot another trash bag out of the cannon.
